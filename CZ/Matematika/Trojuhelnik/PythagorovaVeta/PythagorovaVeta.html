Znění pythagorovy věty

Součet obsahů čtverců nad odvěsnami, je roven ovsahu čtverce nad přeponou.

Vzorečky

a2 = c2 - b2
b2 = c2 - a2
c2 = a2 + b2

Vysvětlivky

a; b; c = označení strany
Přepona = strana ležící naproti pravému úhlu
Odvěsna = strana tvořící pravý úhel
